层次选择器和属性选择器

$(".box p") 等于 $(".box").find("p")    //相当于没有大于号的下一级p

$(".box > p") 相当于 $(".box").children("p")    //相当于绝对下一级不可改变

$(".box + p") 相当于 $(".box").next("p")    //相当同级屁股后的“第一个”

$(".box").nextAll("p")    //.box的同级后所有的P, 中间拦截也没用！

$(".box").nextUntil("p")    //当前box的屁股第一个无限往后，到P停止

$(".box").prev("p")    //.box的同级前“一个”P, 拦截就完蛋，只有前一个有用，前一个没有P就无效
